# Where is Ryan Bigg?

## Contributions welcome!  To get rolling:

Fork and clone https://github.com/wiseleyb/whereisryanbigg

```
cd whereisryanbigg
bundle install
cp config/database.yml.example config/database.yml
... modify database.yml
bundle exec rake db:nuke
```
